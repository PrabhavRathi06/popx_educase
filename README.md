# PopX App 🚀

> A sleek, responsive React + Vite application implementing the PopX UI assignment with local storage authentication.

## 🌟 Features

- **Landing Page**: Engaging entry point for the application.
- **User Authentication**: Secure Sign-up and Login functionality.
- **Local Storage Auth**: Persists user sessions locally.
- **Dynamic Profile Page**: Displays user-specific data dynamically.
- **Logout Functionality**: Safely clears user sessions.
- **Responsive Design**: Mobile-first UI tailored for seamless cross-device experiences.

## 💻 Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Routing**: React Router DOM v7
- **Styling**: Vanilla CSS3

## 🚀 Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

Ensure you have Node.js and npm installed on your system.

### Installation

1. Clone the repository and navigate to the project directory:
   ```bash
   cd popx_educase
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
